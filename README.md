# 享樂酒店訂房平台

## Node Version
```bash
20.10.0
```
## 注意事項
```
所有文字請使用em(px值 / 16)
例：18px -> 1.125em

img alt必須有值
a標籤必須有href, title

目前focus樣式請貼上：focus:block-focus

所有頁面皆已包含h1, h2，往下延伸寫即可 (必須有上一級才能往下寫，例：頁面要寫h4 則必須先確認該頁面包含h1, h2, h3)
```

## `Git`提交規範
  - `feat` 增加新功能
  - `fix` 修復問題/BUG
  - `style` 調整風格且不影響運行結果
  - `perf` 優化/性能提升
  - `refactor` 重構
  - `revert` 撤銷修改
  - `test` 測試相關
  - `docs` 文檔/註釋
  - `chore` 配置檔修改
  - `types` 類型定義文件修改
  - `wip` 開發中尚未完成

## Get Started
```bash
npm install
npm run dev
```

## Production

npm run build
